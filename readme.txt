== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use shortcode in page, post or in widget
4. If you want to use news ticker in your theme function place '<?php echo do_shortcode('YOUR_SHORTCODE');?>' in your templates.

Shortcodes 
<strong>1. General Ticker</strong>
<pre>[ticker_list  id="1"]</pre>

<strong>2. Ticker with custom color and Background-color</strong>
<pre>[ticker_list id="2" color="cyan" backgroundcolor="white"]</pre>

<strong>3. Ticker with Speed Change</strong>
<pre>[ticker_list id="3" text="LATEST NEWS" typespeed="100" speed="1000" backgroundcolor="#428BCA"]</pre>

<strong>4. Ticker with ctaegory ID</strong>
<pre>[ticker_list id="4" text="BREAKING NEWS" typespeed="300" speed="2000" category_slug="category_ID"]</pre>
